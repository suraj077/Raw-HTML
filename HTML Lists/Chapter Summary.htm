Chapter Summary

  1. Use the Html 'ul' element to define an unordered list.
  2. Use the CSS 'list-style-type property to define the list item marker.
  3. Use the HTML 'ol' element to define an ordered list.
  4. Use the HTML 'type' attribute to define the numbering type.
  5. Use the HTML 'li' elememnt to define a list item.
  6. Use the HTML 'dl' element to define a description list.
  7. Use the HTML 'dt' element to define the description term.
  8. Use the HTML 'dd' element to describe the term in a description.
  9. Lists can be nested inside lists
  10. List items can contain other HTML element.
  11. Use the CSS property 'float:left or display:inline to display a list horizontally.
